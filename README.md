# Juego de Arkanoid

Clon de Arkanoid/Breakout hecho con HTML, CSS y JavaScript puro — cero
dependencias (sin build tool, sin bundler, sin package manager).

## Cómo jugar

Abrí `index.html` directamente en el navegador. No hace falta servidor ni
paso de build.

- Mové la pala con el mouse o con las flechas / A-D.
- Lanzá la bola con click o la barra espaciadora.
- Rompé los bloques para sumar puntos y avanzar de nivel.
- Tenés 3 vidas. El juego tiene 3 niveles progresivos, cada uno más rápido
  que el anterior.

## Desarrollo

Este proyecto sigue un flujo spec-driven: las specs viven en `specs/` y se
diseñan y aplican con los comandos `/spec` y `/spec-impl`. Ver `CLAUDE.md`
para el detalle del flujo y del estado actual del proyecto.
